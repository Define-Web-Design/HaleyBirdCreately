# Deep Integration Plan: AI-Driven Mood Capsules

Our goal is to enable effortless emotional storytelling on our platform by having AI dynamically group user content into “Mood Capsules” and generate captivating, mood-based captions. This plan also ensures that vital UI reorganizations—such as moving accessibility tools from the top toolbar to the sidebar—are included in a combined, cohesive update.

## 1. Backend Deep Integration

### A. Enhanced Mood Capsule API Endpoint

**Overview:**  
We’re creating an endpoint (`/api/mood-capsule`) that accepts user content (images, text, etc.), performs deep sentiment analysis (even assigning mixed moods with weighted values), groups content, and generates narrative captions in a dynamic tone (Poetic, Concise, or Conversational).

#### Key Enhancements:
- **Layered Sentiment Analysis:**  
  Detect multiple emotions within a single item—assign weights (e.g., 70% joy, 30% nostalgia) so that mixed moods can be clustered as “Hybrid Moods” (e.g., “Bittersweet Reflections”).
- **Metadata Tagging:**  
  Each content item will be tagged with details like timestamps, engagement metrics, and any user notes to enrich context during grouping.
- **Dynamic Tone Options:**  
  Allow generation of captions in various styles. Provide UI toggles that let the user choose their preferred tone (e.g., Poetic vs. Concise).

#### Code Example (server/api/moodCapsule.js):

```js
// server/api/moodCapsule.js
const express = require('express');
const axios = require('axios');
const router = express.Router();

// Dummy analysis: In practice, integrate with Hugging Face or OpenAI.
async function analyzeContent(contentItems) {
  // Simulate detailed sentiment assignment
  return contentItems.map(item => ({
    ...item,
    mood: 'joyful', // Replace with weighted mood logic
    metadata: item.metadata || {}
  }));
}

function groupContent(analyzedData, moodInput) {
  // Primer: group items by comparing their mood attribute.
  // Enhance this to handle "Hybrid Moods" if needed.
  return [{
    title: moodInput === 'joyful' ? 'Burst of Joy' : 'Reflective Moments',
    subtitle: moodInput === 'joyful'
      ? 'Moments that lit up your week.'
      : 'Thoughtful and serene experiences.',
    items: analyzedData.filter(item => item.mood === moodInput),
    summary: `A series of ${moodInput} experiences`
  }];
}

async function generateCaption(summary, tone) {
  // Example dynamic caption generation using OpenAI's API
  try {
    const response = await axios.post('https://api.openai.com/v1/completions', {
      prompt: `Generate a ${tone} caption for: ${summary}`,
      max_tokens: 50,
      model: 'text-davinci-003'
    }, {
      headers: { Authorization: `Bearer ${process.env.OPENAI_API_KEY}` }
    });
    return response.data.choices[0].text.trim();
  } catch (error) {
    console.error('Caption generation error:', error);
    return "An inspiring moment captured perfectly.";
  }
}

router.post('/mood-capsule', async (req, res) => {
  try {
    const userContent = req.body.content; // Array of content items
    const moodInput = req.body.mood; // e.g., "joyful", "reflective", etc.
    
    // Step 1: Analyze content for layered sentiment.
    const analyzedData = await analyzeContent(userContent);
    
    // Step 2: Group content based on detected mood markers.
    const groupedData = groupContent(analyzedData, moodInput);
    
    // Step 3: Generate a caption for each content group.
    const moodCapsules = await Promise.all(groupedData.map(async group => {
       const caption = await generateCaption(group.summary, "Poetic"); // Tone can be dynamic
       return {
         title: group.title,
         subtitle: group.subtitle,
         caption,
         moodColor: "#FFD700", // Could be dynamic based on mood
         progress: Math.floor(Math.random() * 100), // Example progress metric
         items: group.items
       };
    }));
    
    // Optional: Increase evolution points and update creative history here.
    res.json({ moodCapsules });
  } catch (e) {
    console.error(e);
    res.status(500).json({ error: 'Internal Server Error' });
  }
});

module.exports = router;
```

  const [capsules, setCapsules] = useState<any[]>([]);
  const [loading, setLoading] = useState(true);
  
  const generateCapsule = async () => {
    setLoading(true);
    try {
      // Replace with real user content retrieval logic
      const userContent = [
        { id: 1, thumbnail: '/images/image1.jpg', data: 'Content 1', metadata: { timestamp: '2025-03-27', tags: ['sunset', 'friends'] } },
