# AI-Driven Mood Capsules Implementation Plan

*Empowering effortless storytelling through AI that groups content based on mood and generates meaningful captions.*

---

## **Overview**

### **Vision**
- **Dynamic Content Grouping:**  
  The AI automatically picks up creative content (images, text, etc.) based on the user's current mood or recent activity patterns. It groups these into "Mood Capsules"—collections that reflect the prevailing emotions.
  
- **Effortless Storytelling:**  
  Each Mood Capsule is given an AI-generated caption and narrative that captures and defines the collected emotions, so users don’t have to relive specifics or overthink their feelings.

- **Seamless Integration into UI:**  
  The feature will reside on the Creative Symbiosis dashboard, adopting consistent mood-based color palettes and visual styling. Evolution points and creative history will track capsule creation events.

## **1. Backend Enhancements**

### **A. Create the Mood Capsule API**

In your Replit Node/Express backend, create an endpoint (`/api/mood-capsule`) that processes creativity input and mood data, groups content accordingly, and returns an AI-generated caption for each group.

```js
// File: server/api/moodCapsule.js
const express = require('express');
const router = express.Router();
// Placeholder functions: replace with real implementations / API calls.
const analyzeContent = async (content) => {
  // Process content for sentiment/emotion analysis – use an NLP or image recognition API.
  // Return analyzed data with emotional markers.
  return content.map(item => ({ ...item, mood: 'joyful', summary: 'Happy moments' }));
};

const groupContent = (analyzedData, moodInput) => {
  // Group items based on similar mood markers.
  const groups = [];
  const filtered = analyzedData.filter(item => item.mood === moodInput);
  if (filtered.length) {
    groups.push({
      title: moodInput === 'joyful' ? 'Moments of Joy' : 'Reflections',
      subtitle: `An array of ${moodInput} moments`,
      items: filtered,
      summary: 'Collected highlights'
    });
  }
  return groups;
};

const generateCaption = async (summary, mood) => {
  // Call an AI text-generation service to generate a caption.
  // For demonstration, returning a simple string.
  if (mood === 'joyful') {
    return `A burst of joyful energy that lights up your day.`;
  } else {
    return `A reflective glimpse into your journey.`;
  }
};

router.post('/mood-capsule', async (req, res) => {
  try {
    const userContent = req.body.content; // An array of creative content items.
    const moodInput = req.body.mood; // For example, "joyful" or "reflective".
    
    // Step 1: Analyze the content for emotional markers.
    const analyzedData = await analyzeContent(userContent);
    // Step 2: Group content based on emotional/mood similarity.
    const groupedData = groupContent(analyzedData, moodInput);
    // Step 3: Generate captions for each capsule.
    const moodCapsules = await Promise.all(groupedData.map(async group => {
       const caption = await generateCaption(group.summary, moodInput);
       return {
         title: group.title,
         subtitle: group.subtitle,
         caption,
         items: group.items
       };
    }));
    
    res.json({ moodCapsules });
  } catch (e) {
    console.error(e);
    res.status(500).json({ error: 'Internal Server Error' });
  }
});
module.exports = router;
```

*Instructions:*  
1. Place this file under your backend routes folder (e.g., `server/api/moodCapsule.js`).  
2. Ensure it is correctly imported in your main `server/routes.ts` (or equivalent) file.

## **2. Frontend Enhancements**

### **A. Create the Mood Capsules Component**

In your React project under `client/src/components/dashboard/`, create a new component `MoodCapsules.tsx`:

```tsx
// File: client/src/components/dashboard/MoodCapsules.tsx
import React, { useState, useEffect } from 'react';
import axios from 'axios';

export default function MoodCapsules() {
  const [capsules, setCapsules] = useState<any[]>([]);
  const [loading, setLoading] = useState(true);
  
  // Function to trigger Mood Capsule generation.
  const generateCapsule = async () => {
    setLoading(true);
    try {
      // Fetch user content; this is an example placeholder.
      const userContent = [
        { id: 1, thumbnail: '/path/to/image1.jpg' },
        { id: 2, thumbnail: '/path/to/image2.jpg' }
      ];
      const response = await axios.post('/api/mood-capsule', {
        content: userContent,
        mood: 'joyful'  // Replace with dynamic user input as needed.
      });
      setCapsules(response.data.moodCapsules);
    } catch (e) {
      console.error(e);
    } finally {
      setLoading(false);
    }
  };

  useEffect(() => {
    generateCapsule();
  }, []);

  if (loading) return <div>Gathering your creative moments...</div>;

  return (
    <div className="mood-capsules">
      {capsules.map((capsule, index) => (
        <div key={index} className="capsule-card" style={{ borderColor: '#FFD700', marginBottom: '1rem', padding: '1rem', borderRadius: '8px' }}>
          <h3>{capsule.title}</h3>
          <p>{capsule.subtitle}</p>
          <blockquote>{capsule.caption}</blockquote>
          <div className="capsule-items" style={{ display: 'flex', gap: '0.5rem', flexWrap: 'wrap' }}>
            {capsule.items.map((item: any, i: number) => (
              <img key={i} src={item.thumbnail} alt="creative snippet" style={{ height: '80px', width: '80px', objectFit: 'cover', borderRadius: '4px' }} />
            ))}
          </div>
          <button onClick={() => { /* Add handler for editing/viewing capsule details */ }} style={{ marginTop: '0.5rem' }}>
            Dive In
          </button>
        </div>
      ))}
    </div>
  );
}
```
