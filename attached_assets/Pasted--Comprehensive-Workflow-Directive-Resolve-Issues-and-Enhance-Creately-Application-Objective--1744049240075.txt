# Comprehensive Workflow Directive: Resolve Issues and Enhance Creately Application

## **Workflow Steps**

### **Step 1: Break Circular Dependencies**
#### **Objective**
Resolve circular dependencies between the following files to prevent render loops and structural errors:
- `ThemeContext.tsx` → `touchFeedback.ts`
- `touchFeedback.ts` → `use-mobile.ts`
- Duplicate files: `use-mobile.ts` and `use-mobile.tsx`.

#### **Actions**
1. **Create a Shared Utilities File**:
   - Move haptic feedback logic from `touchFeedback.ts` into a new utilities file:
     ```typescript
     // File: client/src/utils/hapticFeedback.ts
     export const triggerHapticFeedback = () => {
       // Add haptic feedback logic here
     };
     ```
   - Update imports in `ThemeContext.tsx` and `touchFeedback.ts` to reference `hapticFeedback.ts`.

2. **Remove Duplicate Mobile Hook File**:
   - Delete `use-mobile.tsx` and retain `use-mobile.ts` as the single source of truth.
   - Update all imports across the app to use `use-mobile.ts`:
     ```javascript
     import { useMobile } from '../hooks/use-mobile';
     ```

3. **Validate Dependencies**:
   - Test the files for proper functionality without circular imports.
   - Confirm haptic feedback works across components.

### **Step 2: Fix ApplePhotosIntegration Component Issues**
#### **Objective**
Resolve the infinite update loop caused by improper React Query configurations and connection status management.

#### **Actions**
1. **Update useQuery Settings**:
   - Configure queries to use aggressive caching and prevent redundant refetching:
     ```javascript
     useQuery('connectionStatus', fetchConnectionStatus, {
       staleTime: Infinity, 
       cacheTime: Infinity, 
       refetchOnWindowFocus: false, 
       refetchOnMount: false
     });
     ```

2. **Refactor Connection Status Query**:
   - Isolate connection status handling logic into a dedicated function or hook to simplify dependency management.

3. **Validate Fix**:
   - Test the component to ensure the infinite update loop is resolved.
   - Verify connection status updates occur correctly without unnecessary refetches.

#### **Actions**
