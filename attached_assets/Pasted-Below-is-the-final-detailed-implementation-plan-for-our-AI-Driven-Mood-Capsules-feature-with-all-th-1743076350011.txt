# AI-Driven Mood Capsules – Replit Implementation Prompt

## **Feature Overview: AI-Driven Mood Capsules**

### **Vision**
- **Dynamic Content Grouping:**  
  The AI automatically analyzes user-generated content based on recent posts, images, text snippets, and activity signals. It detects the user’s current emotional state (via mood settings and subtle behavioral cues) and groups related content into a “Mood Capsule.”
  
- **Effortless Storytelling:**  
  For each capsule, the AI generates a succinct, emotion-rich caption that encapsulates the user’s feelings and creative vibe. This narrative not only saves mental effort but also encourages authentic online engagement without forcing users to relive every specific memory.
  
- **Seamless Integration:**  
  The feature links closely with our Creative Symbiosis framework. It adapts to the mood-based color palette generator, evolution points, and creative history—providing a consolidated and consistent creative journey.

## **Implementation Plan**

### **1. Backend Enhancements**

#### **A. AI Content Grouping and Captioning Endpoint**
- **Create a New API Endpoint:**  
  - **Route:** `/api/mood-capsules`
  - **Purpose:** Accepts recent user content, current mood settings, and additional metadata (e.g., chosen color palette).  
  - **Technology:** Node.js with Express (or your preferred backend framework on Replit).

- **Steps:**
  1. **Content Analysis:**  
     - Process submitted content items (text, images) using sentiment analysis libraries or external APIs.  
     - Use transformer-based sentiment models (like Hugging Face models) to determine the predominant emotions.
  
  2. **Content Grouping:**  
     - Cluster similar items together that share emotional markers.  
     - Extract key themes using keyword extraction algorithms.
  
  3. **Caption Generation:**  
     - Feed the grouped data to an AI text generation API (like GPT-3/4 if available on your plan) to generate a concise, mood-driven caption for the group.
     - Ensure the generated text is reflective but avoids forcing a user to “re-live” any memory, instead emphasizing a forward, positive narrative.
  
  4. **Response Data:**  
     - Return a JSON object structured as follows:
       ```json
       {
         "moodCapsules": [
           {
             "id": "capsule_1",
             "title": "Joyful Moments",
             "caption": "A burst of joyful energy carrying you through bright days.",
             "contentItems": [ /* array of content IDs or data */ ],
             "suggestedColorPalette": { /* palette details here */ }
           },
           // ... additional capsules
         ]
       }
       ```

#### **B. Integration with Evolution Points and Creative History**
- Award evolution points for each successful generation of a Mood Capsule.
- Record each mood capsule activity into the user’s creative history for future reference.

#### **Example Code Snippet (Express Endpoint):**

```javascript
// server/routes.js (or routes.ts if using TypeScript)
const express = require('express');
const router = express.Router();

// Dummy functions to simulate processing
async function analyzeContent(userContent) {
  // Insert your NLP / sentiment analysis here (or call external APIs)
  return { mood: 'joy', keywords: ['bright', 'energetic'] };
}

async function generateCaption(analysisResult) {
  // Call your AI text generation API (or simulate it for now)
  return `A burst of ${analysisResult.mood} energy carrying you through bright days.`;
}

router.post('/api/mood-capsules', async (req, res) => {
  try {
    const { userContent, moodSettings, colorPalette } = req.body;
    // Analyze the content
    const analysisResult = await analyzeContent(userContent);
    // Group content - for now assume userContent is one group
    const caption = await generateCaption(analysisResult);
    const capsule = {
      id: 'capsule_' + Date.now(),
      title: analysisResult.mood === 'joy' ? 'Joyful Moments' : 'Moments',
      caption,
      contentItems: userContent,
      suggestedColorPalette: colorPalette // Adjust as needed
    };
    // Optionally add evolution points and update creative history here
    res.json({ moodCapsules: [capsule] });
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});

module.exports = router;
```

### **2. Frontend Enhancements**

#### **A. Mood Capsules React Component**

- **File Path:** `client/src/components/dashboard/MoodCapsules.tsx`

#### **Component Requirements:**
1. **Display Capsule Cards:**  
   Each card displays:
   - Thumbnail previews (if applicable) of content items.
   - The AI-generated title and caption.
   - A progress indicator, e.g., "3 out of 5 items in this capsule."
   - A “Dive In” or “Edit” button.
  
2. **User Interaction:**  
   - A persistent "Generate Capsule" button in the sidebar triggers the backend endpoint.
   - Inline editing for title and caption, with a “save” option that updates the capsule.
   - Share functionality: buttons to share via social media or save to the profile.

#### **Example Component (React/TypeScript):**
```tsx
// client/src/components/dashboard/MoodCapsules.tsx
import React, { useState, useEffect } from 'react';
import axios from 'axios';

interface MoodCapsule {
  id: string;
  title: string;
  caption: string;
  contentItems: any[];
  suggestedColorPalette: any;
}

export default function MoodCapsules() {
  const [capsules, setCapsules] = useState<MoodCapsule[]>([]);
  const [loading, setLoading] = useState<boolean>(true);

  useEffect(() => {
    fetchMoodCapsules();
  }, []);

  async function fetchMoodCapsules() {
    try {
      const response = await axios.post('/api/mood-capsules', {
        userContent: [/* Populate user content data */],
        moodSettings: { currentMood: 'joy' },
        colorPalette: { /* user’s palette data */ }
      });
      setCapsules(response.data.moodCapsules);
    } catch (error) {
      console.error('Error fetching mood capsules:', error);
    } finally {
      setLoading(false);
    }
  }

  function renderCapsule(capsule: MoodCapsule) {
    return (
      <div key={capsule.id} className="capsule-card" style={{ borderColor: capsule.suggestedColorPalette?.accent || '#ccc' }}>
        <h3>{capsule.title}</h3>
        <p>{capsule.caption}</p>
        <button onClick={() => alert('Edit functionality coming soon!')}>Edit</button>
        <button onClick={() => alert('Share functionality coming soon!')}>Share</button>
      </div>
    );
  }

  return (
    <div className="mood-capsules-container">
      <h2>Your Mood Capsules</h2>
      {loading && <p>Gathering your creative moments…</p>}
      {!loading && capsules.length === 0 && <p>No capsules generated yet.</p>}
      <div className="capsules-grid">
        {capsules.map(capsule => renderCapsule(capsule))}
      </div>
      <button onClick={fetchMoodCapsules} className="generate-capsule-btn">Generate New Capsule</button>
    </div>
  );
}
```
