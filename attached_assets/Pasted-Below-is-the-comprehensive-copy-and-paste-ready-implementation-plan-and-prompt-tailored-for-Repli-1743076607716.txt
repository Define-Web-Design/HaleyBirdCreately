# AI-Driven Mood Capsules for Creative Symbiosis  
*Empowering effortless emotional storytelling by grouping content based on user mood and generating AI‐powered captions.*

---

## **Feature Overview: AI-Driven Mood Capsules**

### **Vision**
- **Dynamic Content Grouping:**  
  The AI automatically gathers images, text, or creative artifacts based on sentiment signals (from mood settings and activity patterns), creating “Mood Capsules.”
  
- **Effortless Storytelling:**  
  Each capsule receives a curated caption and a narrative description without forcing the user to relive or overanalyze their emotions.

- **Seamless Integration:**  
  Mood Capsules tie into the Creative Symbiosis framework via mood-based color palettes, evolution points, and creative history. This keeps the experience consistent across your platform.

## **Implementation Plan**

### **1. Backend Enhancements**

#### **A. AI Content Grouping and Captioning Endpoint**
- **Create a New API Endpoint:**  
  On Replit (Node.js/Express), add an endpoint `/api/mood-capsule` that receives the user’s creative content, mood data, and optionally color palette info.

- **AI Processing Flow:**
  1. **Content Analysis:**  
     - Use transformer-based sentiment analysis (e.g., Hugging Face models or OpenAI’s API) to extract emotional signals from text and images.
     - Cluster content items with similar emotional markers.
  
  2. **Grouping Strategy:**  
     - Group items and extract key themes/keywords from each cluster.
  
  3. **Caption Generation:**  
     - Call an AI text-generation API to produce a short narrative caption (e.g., "A burst of joyful energy" or "A reflective moment of solace").
  
  4. **Return Structured Data:**  
     - Return JSON with grouped content arrays, each accompanied by a title, a subtitle, and the generated caption.  
     - Optionally, suggest mood-based adjustments to the color palette.

*Example Node.js snippet for Replit:*

```js
// File: server/api/moodCapsule.js
const express = require('express');
const router = express.Router();
// Assume you have functions analyzeContent(), groupContent(), generateCaption() defined
router.post('/mood-capsule', async (req, res) => {
  try {
    const userContent = req.body.content; // Array of content items
    const moodInput = req.body.mood; // e.g., "joyful", "reflective", etc.
    
    // Step 1: Analyze content sentiment
    const analyzedData = await analyzeContent(userContent);
    // Step 2: Group content by mood markers
    const groupedData = groupContent(analyzedData, moodInput);
    // Step 3: Generate caption for each group
    const moodCapsules = await Promise.all(groupedData.map(async group => {
       const caption = await generateCaption(group.summary, moodInput);
       return {
         title: group.title, // e.g., "Moments of Joy"
         subtitle: group.subtitle, // e.g., "An array of happy moments"
         caption,
         items: group.items
       }
    }));
    
    res.json({ moodCapsules });
  } catch (e) {
    console.error(e);
    res.status(500).json({ error: 'Internal Server Error' });
  }
});
module.exports = router;
```

*Note: Replace helper functions (`analyzeContent`, `groupContent`, `generateCaption`) with your chosen AI service integration or custom logic available on Replit using npm packages and appropriate APIs.*

### **2. Frontend Enhancements**

#### **A. Mood Capsules Component**
- **React Component:**  
  Create a new component `MoodCapsules.tsx` under `client/src/components/dashboard/`. This component will:
  - **Display Grouped Content:**  
    Render interactive card views for each capsule. Each card shows a visual summary (thumbnails), a title, and the AI-generated caption.
  - **User Editing Options:**  
    Provide inline editing of the title/caption and allow users to re-order or remove items.
  - **Visual Integration:**  
    Use your existing mood-based color palettes for styling (e.g., dynamic borders that reflect the detected mood).

*Example React Component (simplified):*

```tsx
// File: client/src/components/dashboard/MoodCapsules.tsx
import React, { useState, useEffect } from 'react';
import axios from 'axios';

export default function MoodCapsules() {
  const [capsules, setCapsules] = useState([]);
  const [loading, setLoading] = useState(true);
  
  // Trigger capsule generation
  const generateCapsule = async () => {
    setLoading(true);
    try {
      const response = await axios.post('/api/mood-capsule', {
        content: /* fetch user content */,
        mood: "joyful"  // This could be dynamic based on user input
      });
      setCapsules(response.data.moodCapsules);
    } catch (e) {
      console.error(e);
    } finally {
      setLoading(false);
    }
  };

  useEffect(() => {
    generateCapsule();
  }, []);

  if (loading) return <div>Gathering your creative moments...</div>;

  return (
    <div className="mood-capsules">
      {capsules.map((capsule, index) => (
        <div key={index} className="capsule-card" style={{ borderColor: /* mood-based color */ '#FFD700'}}>
          <h3>{capsule.title}</h3>
          <p>{capsule.subtitle}</p>
          <blockquote>{capsule.caption}</blockquote>
          <div className="capsule-items">
            {capsule.items.map((item, i) => (
              <img key={i} src={item.thumbnail} alt="creative snippet" />
            ))}
          </div>
          <button onClick={() => {/* open capsule details/edit mode */}}>Dive In</button>
        </div>
      ))}
    </div>
  );
}
```
